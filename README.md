## About

GoCalc provides free, fast, browser-based tools that work without signup or installation. All tools run entirely in your browser for maximum privacy and speed.

## Tools

BMI calculator, loan calculator, mortgage calculator, compound interest, and 15+ more.

## Features

- 100% free, no signup required
- Works in any modern browser
- Mobile-friendly responsive design
- Fast loading (under 1 second)
- No data sent to servers — everything runs locally
- Dark mode support
